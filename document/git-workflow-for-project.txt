1. github에 저장소 만들기 ( 팀원 중 1인이 대표 작업 )

   - 저장소 이름은 이클립스의 workspace 이름으로 사용될 것임을 고려하여 작성

   - 팀원 초대

2. 로컬 저장소 만들기 + 서버 연결 ( 팀원 중 1인이 대표 작업 )

   - git clone "저장소 url"
   
   - cd "저장소폴더"

   - gitignore 파일 수정

3. eclipse 실행

   워크스페이스 선택 창에서 워크스페이스 경로를 2에서 clone된 폴더로 지정

4. 이클립스 설정

   - tomcat 런타임 등록

   - web 파일 Encoding 설정 

   - 기타 설정

5. 웹 프로젝트 만들기

   dynamic web project 템플릿 사용

6. 프로젝트 기본 구성

   - 웹 템플릿 파일 복사 등

7. 저장소 업데이트

    git add .

    git commit -m "init project"

    git push origin master

8. 워크스페이스 공유 ( 위 작업을 수행한 팀원을 제외한 나머지 모든 팀원 )

   git clone "저장소 url"

9. 이클립스 실행

   워크스페이스 선택 창에서 워크스페이스 경로를 7에서 clone된 폴더로 지정

10. 이클립스 설정

   - tomcat 런타임 등록

   - web 파일 Encoding 설정 

   - 기타 설정

11. 웹프로젝트 생성

     dynamic web project 템플릿 사용하고 4에서 만든 프로젝트 이름과 동일한 이름으로 생성
